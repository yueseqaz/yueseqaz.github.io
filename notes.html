JAVA学习

grammar

 注释

  单行注释：// 利用ctrl+/实现对当前行注释

  多行注释：/* 利用ctrl+shift+/实现对选中的多行进行注释

  文档注释：/** * */

  注释不会被执行 也不会被编译到class文件中

 字面量：数据在程序中的书写格式

  字符：单引号'' 不能为空 有且只有一个字符 可以是空格

  字符串：双引号括起来 里面可以是任意内容 包括字符串以及字符以及一个或多个空格

  布尔值；true和false

  其它：/n换行 /t 一个tab键的位置 null 空值

 变量：用来存储一个数据的，本质是内存中的一块区域

  变量的优点：使用变量记要处理的数据，是编码更加灵活。管理更加方便

  格式：数据类型 变量名=数字 （数据内心：int整型 double 小数 ）ps：变量里面的数据可被替换

  变量的定义结束是从变量以后最近的一个”}“截止的 就是变量前面的”{‘的另一半，而不是无脑的看变量后面的第一个”}“，还有变量定义什么类型 就提供对应类型的数据 而且必须先声明 而且定义时可以不赋初值 但是第一次使用的时候必须赋予初值

   变量在同一个范围内只能定义一次 有多个变量的话就定义不同的变量名 eg：int a=1；int a=2：在一个范围内就会出现报错

 关键字 标识符

  关键字java已定义

  标识符只能由数字 字母 下划线 "_" 以及美元符号”$" 不能以数字开头 不能使用关键字 大小写敏感 中文也可以 但不提倡 建议英文 首字母小写 类似studentNumber这种

 数据类型

  基本数据类型

   整型

    byte：占用一个字节 八位 范围 -128———137

    short：占用两个字节 16位

    int：占用四个字节 32位

    long：占用八个字节 64位 在赋初始值时 默认是int型 应该在数值后面加L/l

   浮点型

    float 默认是double 需要在数值后面加F/f

    double

   字符型

    char：不能为空 只能一个

   布尔值

    boolean

     true和false

  引用数据类型

   string：字符串型 后面是字符串

idea的使用

 删除当前行利用ctrl+x/y

 ctrl+d：复制当前行至下一行

 ctrl+alt+l：格式化代码

 ctrl+shift+上下方向键移动代码

java概述

 sun公司于1995发布 2009年被甲骨文公司收购

 java之父：詹姆斯.高斯林

 隶属于高级编程语言：介于汇编型语言与解释型语言之间

 java 体系

  Java se：标准版（java的技术和核心

  java ee：企业级：互联网和企业级应用开发( 被认可）

  java me：小型版：移动端 （不被认可）

 jdk

  Java运行时环境（JRE）：用于运行Java程序。 编译器（javac）：将Java源代码编译成字节码。 解释器（java）：运行编译后的Java字节码。 标准类库：提供大量预定义的类和接口，用于常见编程任务。 开发工具：如文档生成器、调试器等。 API文档：提供Java类库的详细使用说明。 示例代码：帮助开发者学习如何使用

  jdk的意思是：java开发工具集成包

创建工程的步骤

 第一步：创建项目(NEW PORJECT)

 第二步：创建模块module（new module）

 第三步{创建包（new package）在模块下的src建立

  包的命名一般是域名倒写加包名（eg：com.baidu.包名）

 第四步：创建类（new class）

path环境变量

 作用：使被添加到环境变量中的·这个软件在命令行中任意位置都能够被执行

 添加步骤：右击此电脑 点击属性 点击高级系统设置 点击环境变量 可选择对当前登陆用户生效还是对此计算机的所有用户生效 然后找到添加的软件的路径 一般是bin目录 然后点击生效即可
