Step 1: Install Git

First, let's check if Git is already installed on your system:

git --version
  • If you see a version number like git version 2.x.x, you're all set! ‚úÖ Ready
  • If Git is not installed, you'll see an installation prompt. Follow the instructions to install it.

Step 2: Configure Git Identity

Set up your Git identity with your GitHub username and email:

git config --global user.name "YourGitHubUsername" git config --global user.email "your.email@example.com"
Important: Use the same email address associated with your GitHub account.

Step 3: Generate SSH Key

SSH keys provide secure authentication between your computer and GitHub.

3.1 Generate the SSH key:

ssh-keygen -t ed25519 -C "your.email@example.com"

Press Enter to accept the default file location. You can skip the passphrase by pressing Enter again (unless you want extra security).

3.2 Copy your public key:

cat ~/.ssh/id_ed25519.pub

Copy the entire output (starts with ssh-ed25519 AAAAC3...)

3.3 Add to GitHub:

  1. Go to GitHub SSH Keys Settings
  2. Click "New SSH key"
  3. Give it a title (e.g., "My Computer")
  4. Paste your public key in the "Key" field
  5. Click "Add SSH Key"

Step 4: Create GitHub Repository
  1. Log in to GitHub
  2. Click the ‚ûï icon in the top right corner
  3. Select "New repository"
  4. Enter a repository name (e.g., my-awesome-project)
  5. Choose visibility (Public or Private)
  6. Click "Create repository"
Tip: Don't initialize with README, .gitignore, or license if you already have a local project.

Step 5: Connect Local Project to GitHub

5.1 Navigate to your project folder:

cd /path/to/your/project

5.2 Initialize Git repository (if not already done):

git init

5.3 Add files to staging area:

git add .

5.4 Create your first commit:

git commit -m "Initial commit"

5.5 Add remote repository:

Copy the SSH URL from your GitHub repository (looks like this):

git@github.com:YourUsername/your-repo-name.git

Then run:

git remote add origin git@github.com:YourUsername/your-repo-name.git

5.6 Push to GitHub:

git push -u origin main
Note: If you're using an older Git version, the default branch might be called master instead of main. In that case, use git push -u origin master.

Step 6: Verify Success

Open your GitHub repository page in a web browser. You should now see all your project files uploaded to GitHub!
